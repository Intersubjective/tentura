# Network source offer

AGPLv3 section 13 requires an operator of a modified Tentura version that
supports remote network interaction to prominently offer every interacting
user an opportunity to receive the Corresponding Source at no charge through a
standard or customary means of software copying.

For a combined GPLv3/AGPLv3 Tentura deployment, the required Corresponding
Source includes the GPLv3-covered work incorporated into the combination as
described by GPLv3 and AGPLv3 section 13.

An operator should provide a clearly visible `Source` link in the remotely
used interface. The link should identify the exact deployed version and lead
to a durable archive or repository containing the complete Corresponding
Source, installation information where required, build scripts, schemas,
migrations, and other material included by the licence definition.

This file is operational guidance. The complete AGPLv3 text controls.
